## 运行说明

``` bash

$ npm install

# serve at localhost:3000
$ node expresstest.js
```

## mongodb Mac相关

``` bash
$ brew -v
$ brew update
$ brew install mongodb
# 新建db
$ sudo mkdir -p /data/db
# chown db
$ sudo chown jiajinpeng /data/db

# mongo控制台
$ mongo

# 启动mongodb
$ mongod
```
